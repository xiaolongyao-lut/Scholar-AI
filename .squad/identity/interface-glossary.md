# Interface Glossary

Use these terms consistently across architecture, implementation, UI, testing, and data work.

- **Literature Folder** — a user-provided directory containing research documents or related artifacts
- **Source Folder** — the concrete folder path chosen for traversal
- **Keyword Filter** — the user-provided keywords used to prune candidate files before deeper extraction
- **Relevance Scan** — the lighter-weight traversal and candidate filtering stage before full extraction
- **Candidate Literature** — files or documents judged relevant enough to continue processing
- **Extraction** — the stage where relevant document content and metadata are pulled into usable context
- **Literature Context** — the retrieved and structured material used to support intelligent dialogue
- **Intelligent Chat** — the dialogue experience grounded in the literature context
- **Insight Message** — a concise literature-grounded inspirational or suggestive line generated from the current literature base
- **Writing Assistant** — later-phase assistance for drafting, review, AI suggestions, and related writing workflows
- **Provenance** — the trace of which file or folder produced a piece of retrieved content
- **Core Path** — the phase-1 workflow: folder input → keyword filter → relevance scan → extraction → intelligent chat
