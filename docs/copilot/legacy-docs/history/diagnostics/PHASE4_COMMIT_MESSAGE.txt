feat (Phase 4): Unified capability registry with approval policies and audit logging

## Overview
Consolidate builtin skills, imported third-party skills, and legacy actions under one safe, 
auditable capability system with comprehensive approval policies and event tracking.

## Changes

### Core Systems Implemented
1. **Unified Skill Registry** (skills/registry.py enhanced)
   - Single registry for builtin and imported capabilities
   - O(1) lookup by ID, filtered queries by mode/kind/source
   - Backward compatible with existing code

2. **Approval Policy System** (skills/approval.py - NEW)
   - ApprovalPolicy enum: AUTO_ALLOWED, REQUIRES_USER_APPROVAL, BLOCKED, GUIDANCE_ONLY
   - CapabilityApprovalProfile: Risk-based approval determination
   - ApprovalStore: Central tracking of requests and decisions
   - Default policies:
     * Builtin skills → AUTO_ALLOWED (low risk)
     * Imported skills → BLOCKED or GUIDANCE_ONLY (high risk)

3. **Audit Logging System** (skills/audit.py - NEW)
   - AuditLog: Event tracking with full replay capability
   - AuditEvent: Immutable event records (9 event types)
   - ExecutionRecord: Individual skill execution tracking
   - Event types: JOB_CREATED, CAPABILITY_RESOLVED, APPROVAL_REQUESTED, APPROVAL_DECIDED,
     EXECUTION_ATTEMPTED, EXECUTION_BLOCKED, EXECUTION_STARTED, EXECUTION_COMPLETED, 
     EXECUTION_FAILED, ARTIFACT_GENERATED, ERROR_OCCURRED

4. **Builtin Skills Loader** (skills/loaders/prompt_builtin_loader.py - NEW)
   - 6 core prompt-backed skills with full descriptors
   - Skills: grammar_checker, paraphrase, tone_adjuster, summarize, expand_details, translate
   - Legacy action mapping for backward compatibility

5. **External Skills Importer** (skills/importers/__init__.py - NEW)
   - Load skills from manifest.json in external directories
   - Automatically disable imported skills by default
   - Set trust_level=LIMITED, script_policy.safe_to_execute=False for imports
   - Comprehensive warning collection

6. **Unified WritingSkillService** (skills/service.py - NEW)
   - Composition root managing registry, approvals, audit log
   - Auto-setup approval profiles per skill source
   - Enforce approval gates on execution
   - Log all critical operations for compliance
   - Maintain legacy action compatibility

### Key Features
- ✅ Imported skills disabled by default (visible but guarded)
