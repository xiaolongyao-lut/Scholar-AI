# Memory Index

This file indexes all memory entries in the auto-memory directory.

## What to Save

Save user preferences, project context, feedback, and decisions that need to persist across conversations.

## File Types

- `user.md` — User role, goals, responsibilities
- `feedback.md` — Guidance on how to approach work
- `project.md` — Ongoing work, goals, initiatives
- `reference.md` — External system pointers
